# Math Trainer — Тренировка счета

## 📱 Описание

Приложение для тренировки устного счета у учеников начальных классов.

**Особенности:**
- 4 типа заданий: сложение, вычитание, умножение, примеры в два действия, утверждения
- Уровни сложности (с каждым уровнем числа растут, время сокращается)
- Таймер на ответ (от 30 до 5 секунд)
- Система жизней (3 ошибки — конец игры)
- Повышение уровня каждые 5 правильных ответов
- Вопросы с ответами "Да"/"Нет"

## 📲 Установка

Скачайте APK-файл из раздела Releases или из корня репозитория и установите на устройство с Android 7.0+.

## 🎮 Как играть

1. Введите ответ в поле (можно отрицательные числа)
2. Нажмите "Проверить"
3. Если вопрос с утверждением — нажмите "ДА" или "НЕТ"
4. У вас 3 жизни и ограниченное время на вопрос
5. Каждые 5 правильных ответов — переход на новый уровень

## 🛠️ Технологии

- Язык: Java
- Среда разработки: Android Studio
- Минимальная версия Android: API 24 (Android 7.0)

## 📁 Структура проекта

- `MainActivity.java` — вся логика игры
- `activity_main.xml` — интерфейс
- `apk/` — установочный файл
